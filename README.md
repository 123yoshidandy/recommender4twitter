# recommender4twitter

## 概要

- 有益なツイートへのアクセス性を重視したTwitterクライアントアプリ。
- 「有益」の指標としてお気に入りやリツイートの数が考えられる。
- TwitterAPIで実現する。
- Webアプリとして作る。
- Chrome Extensionsもありかも。
